    C 的可变参数¶

      使用C语言的可变参数模板，实现类似int printf(const char* format, ...)的函数实现。并且对应LOG(format, ...)宏的定义。

    可变参数函数¶

    函数定义¶

      [type] FuncName([arg1], [arg2], ...)

      [type] : 为函数的返回值类型

      [arg1] [arg2]: 为函数参数，可以有许多个，要实现可变参数座后一个一定是...

    example:¶

    int PrintDoubles(const char* title, int n, ...);

    可变参数的使用¶

      有了上边的函数定义，就可以使用了。

      va_list args 定义一个可变参数列表

      va_start(va_list, arg) 初始化参数列表, arg是在函数参数当中最后一个具有名称的参赛（紧跟...的那个参数）。

      va_arg(va_list, argType)获取参数, 返回该类型的数据，并且每次获取都是下一个参数。

      va_end(va_list)释放参数列表

    example:¶

    #include <stdarg.h>  // 使用可变参数时，要引用头文件

int PrintDoubles(const char* title, int n, ...)
{
    va_list args;
    double arg;
    va_start(args, n;

    // 使用可变参数列表
    arg = va_arg(args, double);

    va_end(args);
}

    相关宏定义¶

      __VA_ARGS__ 在预处理时，会被实际的参数集代替

      args... 在使用时，直接引用名称args即可在预处理替换成参数集

    example¶

    #define LOG(format, ...)       printf(format, __VA_ARGS__)

#define LOG(format, args...)   printf(format, args)

    补充¶

      ##的作用是对token进行连接，如果token为空，则不进行链接，因此可以实施省略可变参数 

    // 可以省略可变参数列表 LOG("Hello world.\n");
#define LOG(format, ...)       printf(format, ##__VA_ARGS__)

#define LOG(format, args...)   printf(format, ##args)
